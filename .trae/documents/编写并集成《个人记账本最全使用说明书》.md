- 编写一份覆盖所有功能点与操作流程的中文使用说明书，面向最终用户，细化到可操作的步骤与注意事项。

# 交付物

- 可选：在“软件设置”页新增“打开使用说明书”入口，一键打开本地说明文件；或新增一个简易内置帮助页。
- 在 `README.md` 增加一行指向该说明书的链接（保持 README 简洁）。

# 说明书结构
1. 简介与快速上手：启动方式、首次运行数据位置、界面布局概览
2. 一级菜单详解：
   - 首页总览：视图切换、图表交互（点击/悬停/滚动）、右侧模块（日度/月度）
   - 账单列表：筛选、排序、列管理、选择统计、编辑/删除/批量操作、详情弹窗
   - 记录账单：支/收/转账录入、类别管理、日期选择、账户选择、键盘区（再记/保存）
   - 账户管理：分组与汇总、编辑模式（内联编辑）、冻结资产、导入/导出与模板
   - 软件设置：菜单布局、数据工具入口、模板与备份、关键词规则（添加/删除/重置）
3. 数据工具（经典布局侧边栏）：导入账单（平台/标准/其它）、智能导入（截图）、导出账单、下载标准模板、备份数据、刷新数据
4. 常见工作流：初次建账、持续记账、分析查看、账户维护、规则优化
5. 进阶技巧与注意事项：重复检测与导出、冻结资产与覆盖导入、列可见性与偏好持久化、鼠标滚动与拖拽、错误提示与恢复（备份/还原）
6. 文件与路径：数据位置、备份文件格式与命名
7. 安全与隐私：不提交密钥、OCR/AI 接入注意事项（如需）
8. 代码参考（可选）：关键功能在代码中的位置（`file_path:line_number`）便于进阶用户查阅
